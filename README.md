# Tech News Scraper
*A beginner project for learning Flask. It scrapes all the top tech news .*

## Packages required
* Flask
* Flask-APScheduler
* Selenium web driver[Chrome]
* Urllib
* Beautiful Soup
* smtplib

It gives the latest top tech news and sends an email notifying the top news via email.
